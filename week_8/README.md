# Food Delivery Receipt Generator

### 1. Purpose of the Application
A simple backend CLI application built to automate customer data collection and calculate breakdown charges (including subtotals, a 5% service charge, and delivery fees) into a formatted receipt.

### 2. Tech Stack
* Python 3

### 3. How to use
1. Make sure all files (`main.py`, `customer.py`, and `receipt.py`) are in the same folder (`week_8`).
2. Run `main.py` in your terminal:
   ```bash
   python main.py
   ```
